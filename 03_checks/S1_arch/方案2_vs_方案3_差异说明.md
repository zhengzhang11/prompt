# 架构方案2 vs 架构方案3 差异说明

> **对比范围**：`02_responses/S1_arch/构架方案2（电机选项优化）.md`（v2.1，57BLF02 + 带宽分离法）与 `构架方案3（被控对象修正）.md`（v1.5，57BLF02 + MO/SO 法）  
> **差异性质**：两方案**电机型号相同（57BLF02）**，核心差异在于 **PI 整定方法**——方案2 使用简化的"零极点对消+带宽分离"，方案3 升级为工业标准"模最优 MO + 对称最优 SO"两步法，并补全了单闭环的完整参数推导。

---

## 1. 触发原因

方案2 完成了电机选型优化（57BLF01→57BLF02），但 PI 整定使用简化的"零极点对消+带宽分离"法，存在以下不足：
1. **未将 PWM 逆变器等效延时纳入被控对象**（仅建模电机 R-L 环节）；
2. 速度环 PI 参数基于经验带宽比，缺乏系统性理论推导；
3. **缺失单闭环的完整被控对象推导和 PI 整定**。

方案3（v1.5）在保持 57BLF02 电机选型的基础上，对控制层进行了全面升级：
1. **被控对象严谨化**：将"PWM 逆变器 + 电机电气 + 机械"三级**完整串联**建模，$T_{PWM,eq}=75\,\mu\text{s}$ 显式引入；
2. **PI 整定升级为工业标准 MO+SO 法**：内环模最优（MO），外环对称最优（SO, $a=4$），增加速度指令前置滤波器；
3. **补全单闭环 PI**：完整的三极点被控对象 + MO 法整定 + 电流截止保护。

## 2. 核心差异对照表

### 2.1 电机与机械参数（两方案一致）

| 条目 | 方案2 | 方案3 | 差异 |
|------|-------|-------|------|
| **电机型号** | 57BLF02 | 57BLF02 | **无差异** |
| **额定功率** | 125 W | 125 W | — |
| **额定力矩 / 峰值力矩** | 0.4 / 1.2 N·m | 0.4 / 1.2 N·m | — |
| **$R_s / L_s$** | 0.30 Ω / 0.75 mH | 0.30 Ω / 0.75 mH | — |
| **$K_t / K_e$** | 0.066 / 6.3 V/krpm | 0.066 / 6.3 V/krpm | — |
| **$J_m / J_{eq}$** | 1.7e-5 / 5.60e-4 | 1.7e-5 / 5.60e-4 | — |
| **$\tau_e = L_s/R_s$** | 2.50 ms | 2.50 ms | — |

### 2.2 被控对象建模（核心差异）

| 条目 | 方案2（简化） | 方案3（严谨） | 差异 |
|------|-------------|-------------|------|
| **PWM 逆变器建模** | ❌ 未显式建模（近似忽略） | ✅ $G_{PWM}(s)=48/(75\mu\text{s}\cdot s+1)$ | **方案3 更严谨** |
| **电流环被控对象** | $G_i=\frac{1/R_s}{\tau_e s+1}$（一阶） | $G_i=\frac{160}{(2.50\text{ms}\cdot s+1)(75\mu\text{s}\cdot s+1)}$（二阶） | 方案3 含延时 |
| **速度环被控对象** | $G_\omega\approx K_t/(J_{eq}s)$（纯积分） | $G_\omega=K_t/[(T_{i,cl}s+1)\cdot J_{eq}s]$（积分+小惯性） | 方案3 显式含电流闭环等效 |
| **单闭环被控对象** | ❌ 未推导 | ✅ $G_{total}=\frac{798.0}{(42.3\text{ms}\cdot s+1)(2.50\text{ms}\cdot s+1)(75\mu\text{s}\cdot s+1)}$ | **方案3 新增** |

### 2.3 PI 整定方法（核心差异）

| 条目 | 方案2（带宽分离） | 方案3（MO+SO） | 差异 |
|------|----------------|--------------|------|
| **整定方法** | 零极点对消 + 带宽分离（经验） | **模最优法（MO）+ 对称最优法（SO）** | **方案3 为工业标准** |
| **理论基础** | 经验带宽比（内环 10× 外环） | 闭环模幅频曲线最平坦（MO）+ 相位裕度最大化（SO） | 方案3 有严格数学推导 |
| **$K_{p,i}$** | 4.71 [V/A]（≈0.098 D*/A 归一化） | **0.1042 [D*/A]**（直接归一化） | 量级接近（±6%） |
| **$K_{i,i}$** | 1880 [V/(A·s)]（≈39.2 归一化） | **41.67 [D*/(A·s)]** | 量级接近（±6%） |
| **$K_{p,\omega}$** | 5.33 [A·s/rad] | **14.14 [A·s/rad]** | 方案3 高 2.65× |
| **$K_{i,\omega}$** | 335 [A/rad] | **5892 [A/rad]** | 方案3 高 17.6× |
| **速度环带宽** | ~100 Hz（628 rad/s） | **~265 Hz**（1667 rad/s，$a=4$） | **方案3 快 2.65×** |
| **前置滤波器** | ❌ 无 | ✅ $\tau_F=2.40\,\text{ms}$（**必须！降超调至~8%**） | 方案3 新增 |
| **单闭环 $K_{p,\omega}^s$** | ❌ 未定义 | **0.01030 [D*/(rad/s)]** | 方案3 新增 |
| **单闭环 $K_{i,\omega}^s$** | ❌ 未定义 | **0.2436 [D*/rad]** | 方案3 新增 |
| **单闭环电流截止** | ❌ 未定义 | ✅ $I_{cutoff}=12\,\text{A}$（必须！） | 方案3 新增 |

### 2.4 PI 参数来源推导链对比

| 环节 | 方案2 推导链 | 方案3 推导链 |
|------|------------|------------|
| **电流环 $K_{p,i}$** | $\tau_i=\tau_e$（对消）→ 经验取 $\omega_{ci}=1/(2T_{PWM})$ | $\tau_i=\tau_e$（对消）→ **MO 条件**：$K_{open}=1/(2T_{PWM,eq})$ → 解析解 |
| **速度环 $K_{p,\omega}$** | 经验带宽比 $\omega_{c\omega}=\omega_{ci}/10$ | **SO 条件**：$a=4$ → $K_{p,\omega}=J_{eq}/(K_t\cdot a\cdot T_{i,cl})$ → 解析解 |
| **速度环超调控制** | 无显式措施 | **前置滤波器** $F(s)=1/(\tau_\omega s+1)$ 对消 PI 零点 |
| **单闭环** | 未推导 | 完整三极点 → PI 对消 $T_m$ → MO 法 → 解析解 |

## 4. 方案3 的技术优势总结

| 维度 | 方案2 → 方案3 改进 |
|------|-------------------|
| **理论严谨性** | 从经验带宽比 → 有严格数学推导的 MO+SO 工业标准方法 |
| **被控对象完整性** | 从单环节电机 R-L → 含 PWM 等效延时的完整三级串联 |
| **速度环带宽** | 100 Hz → **265 Hz**（提升 2.65×），抗扰能力显著增强 |
| **超调控制** | 无显式措施 → **前置滤波器**将 SO 法原生 43% 超调降至 ~8% |
| **并行开发支持** | 无 → S4 可用 `controller_tuning.slx` **独立预调**，不等 S2/S3 |
| **单闭环支持** | 缺失 → 完整的被控对象推导 + PI 整定 + 电流截止保护 |
| **代码即文档** | 无 → §5.3.5 直接给出可复制粘贴的 `init_params.m` 代码段 |
