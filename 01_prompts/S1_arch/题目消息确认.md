## 输入速度函数（MATLAB）

```matlab
function w_ref = fcn(t)
        % 转换系数：rpm 转 rad/s (控制器的标准输入单位)
        rpm2rad = pi / 30;

        % --- AGV 底盘综合测试指令时序 ---
        if t < 1.0
                % 0~1s: 低速靠站起步
                n_cmd = 60;      % 60 rpm
        elseif t < 3.0
                % 1~3s: 切入通道高速巡航 (工况③ 测试: 60 -> 240 阶跃)
                n_cmd = 240;     % 240 rpm
        elseif t < 5.0
                % 3~5s: 减速准备换向 (测试下坡/减速制动斩波是否动作)
                n_cmd = 60;      % 60 rpm
        elseif t < 7.0
                % 5~7s: 紧急反转倒车 (工况④ 测试: 60 -> -240 过零失速风险)
                n_cmd = -240;    % -240 rpm
        else
                % 7s 以后: 刹车停机
                n_cmd = 0;       % 0 rpm
        end

        % 输出物理角速度指令
        w_ref = n_cmd * rpm2rad;
end
```
