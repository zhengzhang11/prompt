## 第 0 章 · 本方案的阅读指南（必读）

本文档按"**分层设计 (Layered Design)**"组织，整个伺服驱动控制系统被分解为 **L1~L6 共 6 层**。每一层对应一位分系统工程师的责任边界、输入契约（上层传下来的量）、输出契约（本层必须交付给下层的量）。**分系统工程师只要遵守契约、不跨层改动，系统就能完整拼装**。

| 层号 | 层名 | 责任人 | 输入契约 | 输出契约 |
|------|------|--------|----------|----------|
| L1 | 工况与负载层 | S2 | 题设 5 个机械参数 + 工况组合 | 阻力力矩 $T_L(v)$、折算惯量 $J_{eq}$、工况时序表 |
| L2 | 电机与传动层 | S2 | L1 输出 | 电机型号、减速比 $i$、电机铭牌参数 $(R_s, L_s, K_e, K_t, J_m, p)$ |
| L3 | 主电路与电源层 | S3 | L2 电机电气参数 + 48 V 母线 | 三相逆变桥拓扑、MOSFET 型号、母线电容、采样电路、保护门限 |
| L4 | 控制器层 | S4 | L2 电机参数 + L3 采样带宽 + 性能指标 | 单闭环 / 双闭环 PI 参数、限幅值、开关频率 |
| L5 | Simulink 集成建模层 | S4 主导 + S2/S3 配合 | L1~L4 全部输出 | 可运行的 `.slx` 模型文件、`params.m` 参数脚本 |
| L6 | 工况验证与报告层 | S5 | L5 模型 | 六大工况仿真数据 + 报告 + PPT |

**关键约束（全员遵守）**：
1. **软件基线**：MATLAB / Simulink **R2023a**，仅使用三个工具箱：**Simulink、Simscape、Simscape Electrical**。**禁用** Motor Control Blockset、SimPowerSystems 遗留库、Stateflow 高级特性（避免 licence / 版本问题）。
2. **参数单一入口**：所有数值参数集中在 `init_params.m` 一个脚本里，模型中**严禁硬编码数字**（仅允许变量名引用）。这是 S5 批量跑工况的前提。
3. **标定方向**：以机器人**前进方向为正**；转速、电流、转矩、母线能量流向正负号在全模型前后一致。
4. **参考模板**：本项目以 MathWorks Simscape Electrical 官方示例 **"BLDC Speed Control"**（自 R2018a 起提供，R2023a 原生支持）为**核心骨架模板**。打开命令：`openExample('simscapeelectrical/BLDCSpeedControlExample')`。详见 2.3 节。

### 2.2 为什么选 BLDC + 双闭环 PI + 三相 MOSFET 桥？（顶层决策仲裁）

| 决策点 | 候选 | 最终选择 | 仲裁理由 |
|--------|------|----------|----------|
| 电机类型 | 直流有刷 / **BLDC** / PMSM / 感应电机 | **BLDC（梯形反电势、霍尔换相）57BLF02 型** | ①57BLF 系列是国内 AGV/机器人底盘**最主流**的 BLDC 规格（ACT/Longs/Promoco/鸣志等 5+ 厂家生产）；②MathWorks 官方 Simscape Electrical 自带 `BLDC` 元件 + 参考模型（`BLDCSpeedControlExample`, `BLDCPositionControlExample`）；③PMSM 需 FOC + SVPWM + Park 变换，**R2023a 标配工具箱下工作量 3 倍**；④有刷直流维护性差，不符合仓储连续作业。**选 24V 额定型号 + 48V 母线 PWM 降压是 MathWorks 模板的标准做法**。 |
| 传动机构 | 减速机 / **行星减速机** / 同步带 | **12.5:1 行星减速机（与 57BLF02 法兰配套）** | 57BL 机座配套行星减速箱市售成熟；效率 0.90 与题设吻合；Simscape `Simple Gear` 可一键建模。 |
| 主电路拓扑 | H 桥 / **三相全桥逆变** / 半桥 | **三相 6-MOSFET 全桥逆变** | BLDC 强制三相；MOSFET 在 48 V / <20 A 场景**远优于 IGBT**（更低 $R_{DS(on)}$、更低开关损耗）。 |
| 调速方式 | 线性调压 / **PWM 调压 + 六步换相** | **PWM + 六步换相（120° 导通）** | BLDC 最成熟方案；`Six-Step Commutation` 逻辑 20 行代码可实现；避免 SVPWM/FOC 的复杂度。 |
| 控制结构 | 开环 / 单闭环 / **双闭环** | **主用双闭环（外速度+内电流）+ 单闭环作对比** | 题目明确要求对比；双闭环能同时满足 R2 (速度超调) 和 R3 (电流超调) 指标；内环限流即是 R8 过载保护。 |
| 位置/速度传感 | 霍尔 / 编码器 / 无感 | **3 × Hall 霍尔传感器** | BLDC 标配；60°/120° 电角度分辨率已满足六步换相；成本极低；Simulink 中用 `Hall Sensor` block 或自定义角度门限实现。 |
| 电流传感 | 分流电阻 / 霍尔电流 | **2 × 低边分流电阻 + 运放** | 48 V / <20 A 场景成熟方案（所有 BLDC 方案 IC 如 DRV8301、MP6540 均内置）。仿真中用理想电流计代替，不影响控制验证。 |
| 制动方案 | 回馈 / **能耗** | **能耗制动（斩波 + 制动电阻）为主，回馈为辅** | 48 V 电池系统回馈存在过充风险；能耗制动简单可靠；R2023a Simscape `Variable Resistor` + 母线电压检测即可实现。 |

#### 3.1.2 数学模型推导（核心，必须完整写进报告 3.1 节）

**【推导 1：单轮线速度与轮侧角速度】**

$$v = n \cdot \frac{2\pi r}{60} \quad \Rightarrow \quad \omega_w = \frac{v}{r} = \frac{2\pi n}{60}$$

代入：
- $n_{low}=60$ rpm → $\omega_{w,low} = 2\pi \approx 6.283$ rad/s → $v_{low} = 0.503$ m/s
- $n_{high}=240$ rpm → $\omega_{w,high} = 8\pi \approx 25.133$ rad/s → $v_{high} = 2.011$ m/s

**【推导 2：平路单轮阻力力矩】**

单轮承受的法向压力 $N_1 = \frac{m g}{2} = \frac{50 \times 9.81}{2} = 245.25$ N
单轮滚动阻力 $F_{r,1} = f \cdot N_1 = 0.04 \times 245.25 = 9.81$ N
轮侧阻力力矩：

$$T_{L,\text{平},w} = F_{r,1} \cdot r = 9.81 \times 0.08 = 0.785 \text{ N·m}$$

**【推导 3：5° 上坡单轮阻力力矩】**（最恶劣工况）

坡上法向压力与切向重力分量：
- $N_{1,\alpha} = \frac{mg}{2}\cos\alpha = 245.25 \times \cos 5° = 244.32$ N
- $F_{g,1} = \frac{mg}{2}\sin\alpha = 245.25 \times \sin 5° = 21.38$ N

单轮滚动阻力（考虑坡度）:
$$F_{r,1,\alpha} = f \cdot N_{1,\alpha} = 0.04 \times 244.32 = 9.77 \text{ N}$$

单轮总切向力（阻力 + 重力分量）：
$$F_{\Sigma,1} = F_{r,1,\alpha} + F_{g,1} = 9.77 + 21.38 = 31.15 \text{ N}$$

轮侧阻力力矩：
$$T_{L,\text{坡},w} = F_{\Sigma,1} \cdot r = 31.15 \times 0.08 = 2.492 \text{ N·m}$$

**【推导 4：减速机折算到电机轴侧 —— 选定减速比 i=12.5】**

选型依据（详见 3.2 节）：57BLF02 额定转速 3000 rpm，故 $i = 3000/240 = 12.5$。

**电机轴侧稳态负载力矩**（考虑传动效率 $\eta_m = 0.90$）：

$$T_{L,m} = \frac{T_{L,w}}{i \cdot \eta_m}$$

- 平路稳态：$T_{L,m,\text{平}} = 0.785 / (12.5 \times 0.9) = 0.0698 \text{ N·m} = 69.8 \text{ mN·m}$
- 坡道稳态：$T_{L,m,\text{坡}} = 2.492 / (12.5 \times 0.9) = 0.2215 \text{ N·m} = 221.5 \text{ mN·m}$

**电机轴侧额定转速**：
$$n_{m,\max} = n_{high} \cdot i = 240 \times 12.5 = 3000 \text{ rpm} \Rightarrow \omega_m = 314.16 \text{ rad/s}$$

**【推导 5：转动惯量折算（机械轴 → 电机轴）】**

整机平动动能等于两个驱动轮的轮侧转动动能之和（能量守恒）：
$$\frac{1}{2} m v^2 = 2 \cdot \frac{1}{2} J_{v,w} \omega_w^2, \quad v = \omega_w r$$

解得**单轮承担的平动等效惯量**：
$$J_{v,w} = \frac{1}{4} m r^2 = \frac{1}{4} \times 50 \times 0.08^2 = 0.08 \text{ kg·m}^2$$

车轮自身转动惯量（估算为均匀圆盘 $m_w \approx 1.5$ kg）：
$$J_w = \frac{1}{2}m_w r^2 = \frac{1}{2}\cdot 1.5 \cdot 0.08^2 = 4.8 \times 10^{-3} \text{ kg·m}^2$$

折算到电机轴侧（除以 $i^2$）：
$$J_{L,m} = \frac{J_{v,w} + J_w}{i^2} = \frac{0.08 + 0.0048}{12.5^2} = 5.43\times 10^{-4} \text{ kg·m}^2$$

**电机自身惯量**（57BLF02 datasheet $J_m = 170$ g·cm² = $1.7 \times 10^{-5}$ kg·m²）：

**系统总折算惯量**：
$$J_{eq} = J_m + J_{L,m} = 1.7\times10^{-5} + 5.43\times10^{-4} \approx 5.60\times10^{-4} \text{ kg·m}^2$$

> **注**：由于 $J_{L,m} \gg J_m$（32 倍），本系统是**典型大负载惯量驱动系统**，对电流环响应速度要求较高。这个结论直接指导 L4 的 PI 带宽设计。

**【推导 6：动态过程所需最大电机轴力矩（R1 指标约束）】**

上升时间 $t_r \le 0.8$ s，从 0 加速到 $\omega_m=314.16$ rad/s，所需角加速度：
$$\dot\omega_m = \omega_m / t_r = 314.16 / 0.8 = 392.7 \text{ rad/s}^2$$

动态加速力矩：
$$T_{\text{加速}} = J_{eq} \cdot \dot\omega_m = 5.60\times10^{-4} \times 392.7 = 0.2199 \text{ N·m}$$

坡道最恶劣工况下电机轴所需**峰值力矩**（加速 + 克服坡道阻力）：
$$T_{m,\text{peak}} = T_{\text{加速}} + T_{L,m,\text{坡}} = 0.2199 + 0.2215 = 0.4414 \text{ N·m}$$

**对比 57BLF02 峰值能力 1.2 N·m：裕量 = 1.2/0.4414 = 2.72 倍 ✓（满足 R1 + R8）**

**【推导 7：电机额定力矩、功率要求】**

稳态最恶劣（坡道巡航）：
$$P_{m,\text{稳态}} = T_{L,m,\text{坡}} \cdot \omega_m = 0.2215 \times 314.16 = 69.6 \text{ W}$$

**结论**：57BLF02 额定 125 W（稳态使用 55.7%）、峰值力矩 1.2 N·m，**动态裕量 2.72 倍 + 坡道稳态裕量 1.81 倍**，完全消除坡道热过载风险，系统选型闭环可行。

### 4.3 关键参数计算

#### 4.3.1 PWM 频率选择

选 **$f_{PWM} = 20$ kHz**（主流 BLDC 驱动频率，超出可闻噪声，MOSFET 开关损耗可控）。

#### 4.3.2 母线电容

纹波电流有效值估算：BLDC 120° 导通，母线电流准方波，$I_{bus,rms} \approx 0.8 I_{phase,peak} = 0.8\times 10 = 8$ A 峰值 → 纹波按 5 A rms 估计。

$$C_{bus} \ge \frac{I_{bus,rms}}{2\pi f_{PWM} \cdot \Delta U} = \frac{5}{2\pi \times 20000 \times 0.48} = 83 \text{ μF}$$

取 **470 μF** 为 5× 裕量，且叠加陶瓷旁路抑制高频尖峰。

#### 4.3.3 制动电阻能量计算（工况⑤关键）

满载 5° 下坡从 240 rpm 减速到 0：
- 平动动能：$\frac{1}{2}m v_{high}^2 = 0.5 \times 50 \times 2.011^2 = 101.1$ J
- 转动动能：$2\times\frac{1}{2}J_{w}\omega_w^2 = 0.0048 \times 25.13^2 = 3.03$ J
- 下坡势能持续释放：$m g v_{high} \sin\alpha = 50 \times 9.81 \times 2.011\times 0.0872 = 86$ W 持续输入
- 摩擦损耗：$2 \times F_{r,1} \times v = 2\times 9.81\times 2.011 = 39.5$ W

制动过程按 1 s 估算，总吸能约 150 J，功率峰值约 200 W。

**制动电阻取 10 Ω / 50W 铝壳电阻**：
- 瞬时功率 $P_{R,peak} = U_{dc}^2/R = 48^2/10 = 230$ W，符合 200 W 峰值要求；
- 铝壳电阻短时过载能力强，50 W 额定 × 3~5 倍短时允许；
- 成本 ~¥ 15。

**斩波门限**：母线电压 $V_{dc} > 54$ V 时开启制动斩波（迟滞控制，回 50 V 关闭）。

### 4.4 保护方案（对应 R10）

| 保护类型 | 硬件实现 | 软件门限 | 动作 |
|----------|----------|----------|------|
| 过流 | 分流电阻 + 比较器快通道 | 相电流 > 12 A 立即封锁 PWM | 硬件 2 μs 内封锁 |
| 过压 | 母线分压 + ADC | $V_{dc} > 58$ V | 软件封锁 + 能耗制动介入 |
| 欠压 | 同上 | $V_{dc} < 40$ V | 软件封锁，防电池深放 |
| 堵转 | 速度反馈为零但电流大 | $\omega < 5$ rad/s 且 $I > I_N$ 持续 0.5 s | 软件封锁 |
| 失速 | 速度指令与实际差持续超阈 | $\|\omega^* - \omega\| > 50\%$ 持续 1 s | 软件封锁 |
| 过温 | MCU 温度传感器 | $T > 85$ °C | 降额运行或封锁 |

### 4.5 Simulink 实现要点

- **MOSFET/IGBT 桥**：Simscape Electrical > Semiconductors > `MOSFET` ×6，参数用 datasheet 给定的 $R_{DS(on)}, V_{DS}, Q_g$。为**加速仿真可退化为** `N-Channel MOSFET` 平均模型或直接用 Simulink `Controlled Voltage Source` + 换相逻辑信号（推荐首次跑通用平均模型，第二轮再换开关模型）。
- **母线电容**：`Capacitor` block, 初值设 48 V 避免启动冲击。
- **分流电阻电流采样**：`Current Sensor` 直接读相电流，省去分流电阻物理建模（减少刚性）。
- **制动斩波**：`MOSFET` + `Resistor`，触发逻辑用 `Compare to Constant` (母线>54V) + `S-R Flip-Flop` 实现迟滞。
- **仿真求解器**：ode23tb，变步长，`Max step = 1e-5`，`Min step = 1e-8`，避免在 20 kHz PWM 下步长爆炸。

```
       ω*          i_q*                     Duty
ω* ──→[─]──→[PI_ω]──→[限幅±I_max]──→[─]──→[PI_i]──→[换相+逆变+电机+机械]──→ω
       ↑                              ↑                │                     │
       └─────────────── ω 反馈 ───────┼────────────────┘                     │
                                       └──────── i_q 反馈 ────────────────── │
                                                                              │
    (i_q 反馈：由两相分流电阻采样，结合换相相位重构为等效 q 轴电流)
```

- **优点**：内环天然限流 = R8 过载保护 + R3 电流超调可控；外环只管转速，动态快；抗扰能力强（扰动主要进内环）。
- **缺点**：两套 PI 4 个参数；需带宽分离（内环带宽 > 外环 5~10 倍）。
- **对 BLDC 的简化**：由于六步换相下每一刻只有两相导通，**"等效 q 轴电流"≈ 母线电流 ≈ 两相电流中非零者**。可用 `max(abs(i_a), abs(i_b), abs(i_c))` 简化，避免引入 Park 变换。

### 5.2 串联被控对象的严谨数学建模（v1.4 并行开发版）

> ⚠️ **v1.4 重要演进**：v1.3 虽已把 PWM 延迟写进被控对象，但**PI 整定仍用"零极点对消 + 简单带宽分离"**，本质上没有利用电流环已经闭环后"小惯性近似"的结论。v1.4 升级到**工业标准的"模最优法（MO）+ 对称最优法（SO）"组合**：既严格处理了 PWM 延迟，又让 S4 可以在电机/电力电子组未交付前用传递函数块**独立预调**，联调误差 <5%。

#### 5.2.1 被控对象的完整信号流（严格按物理顺序串联）

并行开发下 S4 需要搭建的预调被控对象 = **PWM 逆变器 + 电机电气 + 机械**三级串联：

```
  [速度 PI] ──i*──→ [电流 PI] ──D*──→ [PWM 逆变器] ──U_eq──→ [电机 R-L 回路] ──i──┐
                                       G_PWM(s)            G_RL(s)                 │
                                                                                    ↓
                                                                        [Kt]·i = Te │
                                                                                    ↓
    ω ←─── [机械负载 1/(J_eq·s+B_m)] ←── Te-TL ─── [负载 T_L(ω,α,m)]              │
    │             G_mech(s)                                                         │
    └──→ 反馈至速度环                                                                │
                                                                                    │
    i ──→ 反馈至电流环                                                                │
                                                                                    │
    e_b = Ke·ω ─（作为电流环扰动，进入 R-L 回路电压平衡方程）─────────────────────┘
```

**关键约定：控制器输出信号的明确物理定义**（S4 必须严格遵守，全组统一）

| 信号 | 符号 | 物理意义 | 单位 | 范围（限幅） |
|------|------|----------|------|--------------|
| 速度环 PI 输出 | $i^*$ | 内环电流参考 | A | $\pm I_{peak}=\pm 12$ A |
| 电流环 PI 输出 | $D^*$ | **PWM 占空比**（双极性） | 无量纲 | $[-1, +1]$ |
| PWM 逆变器输出 | $U_{eq}$ | 电机回路等效端电压 | V | $D^* \cdot U_{dc}$ |

> **为什么电流环 PI 输出定义为占空比 $D^*$ 而不是电压 $U^*$**：占空比在 $[-1,+1]$ 内有**固定物理意义**（-1=全反，0=停，+1=全正），限幅直接对应逆变器饱和，不随母线电压变化。这是 TI、ST、Microchip 所有工业电机驱动器 IP 的标准做法。

#### 5.2.2 环节 1：PWM 逆变器 $G_{PWM}(s)$

三相 MOSFET 逆变器在平均模型下是"**带延时的放大器**"：

**稳态增益**：
$$K_{PWM} = U_{dc} = 48\ \text{V}$$

（占空比 $D^*=\pm 1$ 对应电机端电压 $\pm U_{dc}$，线性映射）

**等效延时**（工程经典推导 Åström-Wittenmark）：
$$T_{PWM,eq} = 1.5 \cdot T_{PWM} = 1.5 \times \frac{1}{20\,\text{kHz}} = 75\ \mu\text{s}$$

（含义：PWM 零阶保持延时 $0.5 T_{PWM}$ + 数字计算一拍延时 $1.0 T_{PWM}$）

工程中将纯延时用一阶惯性等效（$\omega < 1/T_{PWM,eq}$ 时误差 <5%）：
$$\boxed{G_{PWM}(s) = \frac{K_{PWM}}{T_{PWM,eq}\cdot s + 1} = \frac{48}{75\,\mu\text{s}\cdot s + 1}}$$

#### 5.2.3 环节 2：BLDC 电机电气回路 $G_{RL}(s)$（两相导通等效直流电机）

BLDC 在 120° 导通模式下，任意时刻**只有两相串联导通**，第三相断开。以 datasheet 给出的**线-线参数**建模（这正对应两相串联回路的等效参数，不需要再乘 2）：

| 参数 | 线-线值 | 数值 |
|------|---------|------|
| 回路等效电阻 | $R_{eq}$ | 0.30 Ω |
| 回路等效电感 | $L_{eq}$ | 0.75 mH |

回路电压平衡方程（反电势 $e_b$ 作为扰动先分离出去，方法见 5.2.6）：
$$U_{eq} = R_{eq} \cdot i + L_{eq}\frac{di}{dt} + e_b, \qquad e_b = K_e \cdot \omega_m$$

从电机端电压 $U_{eq}$ 到回路电流 $i$ 的传递函数：
$$\boxed{G_{RL}(s) = \frac{I(s)}{U_{eq}(s)} = \frac{1/R_{eq}}{\tau_e\cdot s + 1},\quad \tau_e = \frac{L_{eq}}{R_{eq}} = \frac{0.75\,\text{mH}}{0.30\,\Omega} = 2.50\ \text{ms}}$$

代入数值：$G_{RL}(s) = \dfrac{3.333}{2.50\,\text{ms}\cdot s + 1}$ [A/V]

#### 5.2.4 环节 3：电流环完整被控对象 $G_i(s)$（S4 预调用）

串联 $G_{PWM}$ 和 $G_{RL}$：

$$\boxed{G_i(s) = \frac{I(s)}{D^*(s)} = G_{PWM}(s)\cdot G_{RL}(s) = \frac{U_{dc}/R_{eq}}{(\tau_e s + 1)(T_{PWM,eq} s + 1)} = \frac{160}{(2.50\,\text{ms}\cdot s + 1)(75\,\mu\text{s}\cdot s + 1)}}$$

**特征分析**：
- 主导时间常数 $\tau_e = 2.50$ ms
- 小惯性时间常数 $T_{PWM,eq} = 75$ μs
- **两者相差 33.3 倍** → 标准的"**一大一小两时间常数**"结构 → **模最优法（MO）的理想场景**

#### 5.2.5 环节 4+5：机械环节 + 速度环完整被控对象 $G_\omega(s)$

力矩方程 + 牛顿方程（负载力矩 $T_L$ 作扰动处理）：
$$T_e = K_t \cdot i, \qquad T_e - T_L = J_{eq}\frac{d\omega_m}{dt} + B_m \omega_m$$

机械传递函数：
$$G_{mech}(s) = \frac{K_t}{J_{eq}\cdot s + B_m} \approx \frac{K_t}{J_{eq}\cdot s} = \frac{0.066}{5.60\times 10^{-4}\cdot s} = \frac{117.9}{s}$$

（阻尼 $B_m \sim 10^{-4}$ 量级，相对 $J_{eq}\cdot s$ 可忽略，纯积分环节）

电流内环按 5.3.1 节模最优法整定后，**闭环等效为一阶小惯性**（这是 MO 法的经典结果，直接给出）：
$$G_{i,cl}(s) \approx \frac{1}{T_{i,cl}\cdot s + 1},\quad T_{i,cl} = 2\cdot T_{PWM,eq} = 150\ \mu\text{s}$$

故**速度环完整被控对象**：
$$\boxed{G_\omega(s) = \frac{\Omega_m(s)}{I^*(s)} = G_{i,cl}(s)\cdot G_{mech}(s) = \frac{K_t}{(T_{i,cl}\cdot s + 1)\cdot J_{eq}\cdot s} = \frac{117.9}{(150\,\mu\text{s}\cdot s + 1)\cdot s}}$$

**特征分析**：
- 一个纯积分（机械惯性）+ 一个小惯性（电流内环闭环等效）
- 这是 **对称最优法（SO）的标准适用场景**

#### 5.2.6 反电势扰动的处理策略

反电势 $e_b = K_e\cdot\omega_m$ 对电流环是**缓慢扰动**（变化频率 $\sim \omega_{cω}$，远低于电流环带宽）。处理方法：

| 方法 | 实现方式 | 效果 | 本方案选择 |
|------|----------|------|------------|
| **A. PI 自带补偿** | 什么都不做，靠 PI 的积分作用缓慢抵消 | 电流环响应略慢（速度阶跃切换时电流有 5~10% 额外动态误差） | ✅ **默认（实现最简）** |
| **B. 反电势前馈** | 在电流 PI 输出后叠加 $D_{ff} = K_e\cdot\omega_m / U_{dc}$ | 电流环动态与 ω 解耦，响应更快 | ⭐ **备选**（若工况③④性能不达标再启用） |

> **S4 执行顺序**：先用方法 A 跑通全部 6 工况；若工况③（高/低速切换）或④（正反转）的电流动态超标，再启用方法 B（1 个加法器 + 1 个乘法器即可）。

### 5.3 PI 参数整定（v1.4：模最优 MO + 对称最优 SO 两步法）

> 📌 **S4 独立预调工作流（并行开发核心）**：
>
> S4 在 S2（电机铭牌）和 S3（逆变器细节）交付前，搭建**独立预调测试模型 `controller_tuning.slx`**，被控对象用 Simulink 原生 `Transfer Fcn` 块按 5.2 节传递函数搭建：
>
> ```
> ┌─ 速度 PI ─┬─→ 限幅 ─→ 电流 PI ─┬─→ G_PWM(s) ─→ G_RL(s) ─┬─→ Kt ─→ 1/(J·s) ─→ ω
> │           │                     │                          │                        │
> │     [ω_ref]                 [i_fb]                    [U_eq]                   [i_fb]
> │                                                                                      │
> └── ω_fb ◄──────────────────────────────────────────────────────────────────────────────┘
>
> 其中：
>   G_PWM(s) = 48 / (75e-6·s + 1)       ← PWM 逆变器
>   G_RL(s)  = 3.333 / (2.50e-3·s + 1)  ← 电机电气
>   Kt = 0.066,  J_eq = 5.60e-4          ← 机械参数
> ```
>
> **并行开发价值**：
> - S4 用本节 MO+SO 公式算出 PI 初值后，直接在 `controller_tuning.slx` 跑 6 个工况波形；
> - S2/S3 完成后，替换成 `BLDCSpeedControlExample` 真实模型，**PI 初值直接复用，联调偏差 <5%**；
> - S4 的预调工作**不会白做**，最多只需 ±15% 范围的现场微调。
>
> **交付物**：`controller_tuning.slx` + 6 个预调波形截图。

#### 5.3.1 内电流环 PI —— 模最优法（Modulus Optimum, MO）

PI 控制器：
$$C_i(s) = K_{p,i}\cdot \frac{\tau_i\cdot s + 1}{\tau_i\cdot s}$$

**第一步：零极点对消（消除电机电气主极点）**

令 $\tau_i = \tau_e = L_{eq}/R_{eq} = 2.50$ ms，PI 的零点 $-1/\tau_i$ 正好对消电机极点 $-1/\tau_e$。

对消后开环传递函数：
$$L_i(s) = C_i(s)\cdot G_i(s) = \frac{K_{p,i}\cdot U_{dc}/R_{eq}}{\tau_i\cdot s}\cdot \frac{1}{T_{PWM,eq}\cdot s + 1} = \frac{K_{open}}{s\cdot(T_{PWM,eq}\cdot s + 1)}$$

其中 $K_{open} = \dfrac{K_{p,i}\cdot U_{dc}}{R_{eq}\cdot \tau_i}$

**第二步：模最优条件 → 闭环阻尼 ζ = 0.707**

对 $L(s) = \dfrac{K_{open}}{s(Ts+1)}$ 形式，MO 法要求 $K_{open} = \dfrac{1}{2T}$，即：

$$\frac{K_{p,i}\cdot U_{dc}}{R_{eq}\cdot \tau_i} = \frac{1}{2\cdot T_{PWM,eq}}$$

解出：
$$\boxed{K_{p,i} = \frac{R_{eq}\cdot \tau_i}{2\cdot U_{dc}\cdot T_{PWM,eq}} = \frac{0.30 \times 2.50\,\text{ms}}{2 \times 48 \times 75\,\mu\text{s}} = 0.1042\ \text{[占空比/A]}}$$

$$\boxed{K_{i,i} = \frac{K_{p,i}}{\tau_i} = \frac{0.1042}{2.50\,\text{ms}} = 41.67\ \text{[占空比/(A·s)]}}$$

**第三步：性能预测（MO 法固有性质，S4 用来自检预调结果）**

| 指标 | MO 固有值 | 本方案数值 |
|------|-----------|-----------|
| 阶跃超调 | 4.3% | 预期 <5% ✓ 满足 R3 |
| 上升时间 | $\approx 4.7\cdot T_{PWM,eq}$ | $\approx 0.35$ ms |
| 闭环带宽 ω_bi | $\approx 1/(\sqrt{2}\cdot T_{PWM,eq})$ | $\approx 9400$ rad/s ≈ **1.5 kHz** |
| 相对阻尼 ζ | 0.707 | — |
| 相位裕度 PM | ≈ 63° | 非常稳健 |

**第四步：闭环等效为一阶小惯性（给外环用）**

MO 法的经典结论：电流环闭环在机械环节感兴趣的频带内近似为
$$G_{i,cl}(s) \approx \frac{1}{T_{i,cl}\cdot s + 1},\quad T_{i,cl} = 2\cdot T_{PWM,eq} = 150\ \mu\text{s}$$

这就是 5.2.5 节速度环被控对象中使用的 $T_{i,cl}$ 的来源。

#### 5.3.2 外速度环 PI —— 对称最优法（Symmetric Optimum, SO）

速度环被控对象 $G_\omega(s) = \dfrac{K_t}{(T_{i,cl} s + 1)\cdot J_{eq}\cdot s}$ —— **积分 + 小惯性**，是 SO 法的标准场景。

**SO 法的设计思想**：在穿越频率 $\omega_c$ 周围对称分布 PI 的零点和小惯性极点，使相位裕度最大化。设参数 $a \geq 2$（对称度）：

$$\boxed{\tau_\omega = a^2\cdot T_{i,cl},\qquad K_{p,\omega} = \frac{J_{eq}}{K_t\cdot a\cdot T_{i,cl}},\qquad K_{i,\omega} = \frac{K_{p,\omega}}{\tau_\omega}}$$

**$a$ 的工程选择**：

| $a$ 值 | 相位裕度 PM | 带宽 $\omega_{c\omega}$ | 阶跃超调（无前置滤波） | 加前置滤波后超调 |
|-------|-------------|-------------------------|-----------------------|----------------|
| 2 | 37° | 3333 rad/s (531 Hz) | 72% | ~20% |
| **4** | **53°** | **1667 rad/s (265 Hz)** | **43%** | **~8%** ✓ |
| 6 | 61° | 1111 rad/s (177 Hz) | 30% | ~5% |
| 8 | 66° | 833 rad/s (133 Hz) | 23% | ~4% |

**本方案选 $a = 4$**（带宽与鲁棒性最平衡；若仿真实测超调超标，退到 $a = 6$）。

**PI 参数计算**（代入 $a=4, T_{i,cl}=150\,\mu\text{s}, J_{eq}=5.60\times 10^{-4}, K_t=0.066$）：

$$\boxed{\tau_\omega = 4^2 \times 150\,\mu\text{s} = 2.40\ \text{ms}}$$

$$\boxed{K_{p,\omega} = \frac{5.60\times 10^{-4}}{0.066 \times 4 \times 150\,\mu\text{s}} = 14.14\ \text{[A·s/rad]}}$$

$$\boxed{K_{i,\omega} = \frac{K_{p,\omega}}{\tau_\omega} = \frac{14.14}{2.40\,\text{ms}} = 5892\ \text{[A/rad]}}$$

#### 5.3.3 SO 法的必备补丁：速度指令前置滤波器

SO 法原生阶跃超调 43%（$a=4$），**直接不满足** R2（<10%）。工程标准做法是在速度参考输入端加**一阶前置滤波器**：

$$F(s) = \frac{1}{\tau_F\cdot s + 1}, \quad \tau_F = \tau_\omega = 2.40\ \text{ms}$$

数学机理：$F(s)$ 正好对消 PI 的零点，使阶跃响应不再有微分作用。加上后**阶跃超调降至 ~8%**（满足 R2 要求）。

**Simulink 实现**：在速度指令 $\omega^*$ 进入误差比较器前串联一个 `Transfer Fcn = [1]/[2.4e-3, 1]`。**这不会影响抗扰性能**，因为负载扰动不经过这个滤波器。

#### 5.3.4 PI 参数整定分步法（S4 在 `controller_tuning.slx` 上执行）

| 步骤 | 动作 | 期望现象 | 若不满足怎么办 |
|------|------|----------|----------------|
| **Step 0** 搭建预调模型 | 用 Transfer Fcn 按 5.2.1 节结构搭建预调平台 | 开环测试：给 D*=0.2 恒值，电流 i 稳态 ≈ $0.2 \times 48/0.30 = 32$ A | 检查传函数值 |
| **Step 1** 内环孤立调试 | 断开外环，内环输入 $i^*=2$ A 阶跃 | 上升时间 <0.5 ms，超调 <5% | $K_{p,i}$ ±50% 迭代 |
| **Step 2** 内环加限幅 + 抗积分饱和 | 加入 $\pm 12$ A 限幅 + back-calculation | 饱和时 $K_{p,i}$ 输出立即饱和而积分不爆炸 | $K_{aw} = 1/K_{i,i}$ |
| **Step 3** 外环加入，空载速度阶跃（带前置滤波器） | 从 0 阶跃到 240 rpm（≈25 rad/s 轮速，对应 ω_m=314 rad/s） | $t_r$ <0.3 s, $\sigma_n < 10\%$ | $a$ 从 4 增到 6 |

| **Step 6** 指令切换 + 四象限 | 60→240→-240 rpm 序列 | 无失控、电流不超 ±12 A | 检查限幅、anti-windup |

| 步骤 | 动作 | 期望 |
|------|------|------|

#### 5.3.5 完整参数清单（v1.4，直接录入 `init_params.m`）

```matlab
%% ===== v1.4 PI 参数（模最优 MO + 对称最优 SO 严谨推导） =====
% 设计依据：电机 57BLF02 + 48V 母线 + 20kHz PWM + 12.5:1 减速
% 被控对象完整模型已含 PWM 逆变器（75μs 等效延时）
% S4 在 controller_tuning.slx 上的预调值，联调偏差 <5%

%% 电力电子装置参数
Udc        = 48;              % V 母线电压
fpwm       = 20e3;            % Hz PWM 频率
Tpwm_eq    = 1.5/fpwm;        % = 75 μs  PWM等效延时
Kpwm       = Udc;             % PWM 稳态增益

%% 电机参数（57BLF02 线-线等效）
R_eq       = 0.30;            % Ω
L_eq       = 0.75e-3;         % H
Kt         = 0.066;           % N·m/A
Ke         = 6.3/1000*60/(2*pi); % V·s/rad（= 0.06016，由 6.3 V/krpm 换算）
tau_e      = L_eq/R_eq;       % = 2.50 ms

%% 机械参数
J_eq       = 5.60e-4;         % kg·m² 折算总惯量
B_m        = 1e-4;            % N·m·s/rad 估算阻尼

%% === 电流环 PI (MO 法) ===
tau_i      = tau_e;                             % = 2.50 ms (对消电机极点)
Kp_i       = (R_eq*tau_i)/(2*Udc*Tpwm_eq);      % = 0.1042 [占空比/A]
Ki_i       = Kp_i/tau_i;                        % = 41.67  [占空比/(A·s)]
I_max      = 12;                                % A 电流限幅
Ti_cl      = 2*Tpwm_eq;                         % = 150 μs 电流环闭环等效时间常数

%% === 速度环 PI (SO 法, a=4) ===
a_SO       = 4;                                 % 对称最优法参数
tau_w      = a_SO^2 * Ti_cl;                    % = 2.40 ms
Kp_w       = J_eq/(Kt*a_SO*Ti_cl);              % = 14.14 [A·s/rad]
Ki_w       = Kp_w/tau_w;                        % = 5892  [A/rad]

%% === 速度指令前置滤波器（必需，降超调至 ~8%）===
tau_F      = tau_w;                             % = 2.40 ms

%% 抗积分饱和 back-calculation 增益
K_aw_i     = 1/Ki_i;
K_aw_w     = 1/Ki_w;
```

### 5.4 单闭环速度控制：被控对象建模与 PI 整定（v1.4 补全）

> ⚠️ **v1.4 之前的方案遗漏了单闭环的被控对象推导和参数整定**。题目明确要求"单闭环与双闭环对比"，本节正式补全，使 S4 能同时交付 `bldc_singleloop.slx`（单闭环对照模型）和 `bldc_diffdrive_top.slx`（双闭环主模型）。

#### 5.4.1 单闭环与双闭环的本质区别

| 对比维度 | 双闭环（5.2~5.3） | 单闭环（本节） |
|----------|-------------------|----------------|
| 控制信号流 | ω* → 速度PI → i* → 电流PI → D* → 逆变器 → 电机 | **ω* → 速度PI → D* → 逆变器 → 电机** |
| 被控对象 | 分两级：内环 G_i(s)，外环 G_ω(s) | **整体一个 G_total(s)（D* 直接到 ω）** |
| 反电势处理 | 内环近似为扰动，外环不见 | **反电势直接耦合在被控对象中** |
| 电流限制 | 内环积分限幅，天然限流 | **必须外加软件电流截止** |

#### 5.4.2 单闭环完整被控对象推导（精确，含反电势）

从物理方程出发（PWM 以外的部分，$T_L = 0$，反电势不忽略）：

$$\text{电路：} U_{eq} = R_{eq} i + L_{eq}\frac{di}{dt} + K_e\omega_m \tag{1}$$

$$\text{机械：} K_t i = J_{eq}\frac{d\omega_m}{dt} + B_m\omega_m \tag{2}$$

Laplace 变换，由 (2) 得 $I(s) = \dfrac{J_{eq}s + B_m}{K_t}\cdot\Omega(s)$，代入 (1)：

$$U_{eq}(s) = \frac{(R_{eq} + L_{eq}s)(J_{eq}s + B_m)}{K_t}\cdot\Omega(s) + K_e\Omega(s)$$

（忽略 $B_m$ 项，因 $B_m \ll K_t K_e / R_{eq}$）整理后：

$$\boxed{G_{motor}(s) = \frac{\Omega_m(s)}{U_{eq}(s)} = \frac{U_{dc}\cdot K_t}{L_{eq}J_{eq}s^2 + R_{eq}J_{eq}s + K_tK_e} = \frac{K_0}{T_eT_m s^2 + (T_e+T_m)s + 1}}$$

其中三个关键参数：
$$K_0 = \frac{U_{dc}}{K_e} = \frac{48}{0.06016} = 798.0\ \text{rad/s} \quad\text{（满占空比理论空载转速）}$$
$$T_m = \frac{R_{eq}J_{eq}}{K_tK_e} = \frac{0.30 \times 5.60\times10^{-4}}{0.066 \times 0.06016} = 42.3\ \text{ms} \quad\text{（机械时间常数）}$$
$$T_e = \frac{L_{eq}}{R_{eq}} = 2.50\ \text{ms} \quad\text{（电气时间常数，与双闭环相同）}$$

**二阶系统特征分析**：

$$\omega_n = \frac{1}{\sqrt{T_eT_m}} = \frac{1}{\sqrt{2.50\times10^{-3}\times 42.3\times10^{-3}}} = 97.2\ \text{rad/s}\quad(15.5\ \text{Hz})$$

$$\zeta = \frac{T_e + T_m}{2\sqrt{T_eT_m}} = \frac{44.8\text{ ms}}{2\times 10.28\text{ ms}} = 2.18 \gg 1\ \text{（强过阻尼）}$$

> **关键物理结论**：$\zeta = 2.18 \gg 1$，系统是**强过阻尼二阶系统**，等价于两个实极点：
> $$p_1 = -\frac{1}{T_e} = -400\ \text{rad/s}（\text{快极点，电气}），\quad p_2 = -\frac{1}{T_m} = -23.6\ \text{rad/s}（\text{慢极点，机械}）$$
> 两极点相差 **16.9 倍**，可以分解为两个一阶串联近似：
> $$G_{motor}(s) \approx \frac{K_0}{(T_ms+1)(T_es+1)}$$

**含 PWM 逆变器的完整单闭环被控对象**（$G_{total}$ = D* → ω）：

$$\boxed{G_{total}(s) = G_{PWM}(s)\cdot G_{motor}(s) = \frac{K_0}{(T_ms+1)(T_es+1)(T_{PWM,eq}s+1)} = \frac{798.0}{(42.3\text{ms}\cdot s+1)(2.50\text{ms}\cdot s+1)(75\mu\text{s}\cdot s+1)}}$$

#### 5.4.3 单闭环 PI 整定策略（PI 零点对消主导慢极点 + MO 法）

**三个极点的频率分析**：

| 极点 | 时间常数 | 对应频率 | 角色 |
|------|----------|----------|------|
| 机械 $-1/T_m$ | 42.3 ms | **23.6 rad/s** | **主导慢极点，限制系统带宽** |
| 电气 $-1/T_e$ | 2.50 ms | 400 rad/s | 次极点，远离工作频带 |
| PWM $-1/T_{PWM}$ | 75 μs | 13333 rad/s | 高频，影响最小 |

**整定策略**：

**第一步**：PI 零点对消主导慢极点（机械时间常数）

$$\tau_{PI} = T_m = 42.3\ \text{ms}$$

对消后，开环传递函数变为：
$$L(s) = C(s)\cdot G_{total}(s) = K_p\cdot\frac{K_0/\tau_{PI}}{s\cdot(T_es+1)\cdot(T_{PWM,eq}s+1)}$$

合并两个剩余小惯性：
$$T_{small} = T_e + T_{PWM,eq} = 2.50\text{ ms} + 0.075\text{ ms} = 2.575\ \text{ms}$$

**第二步**：MO 法确定 $K_p$（与双闭环电流环同理）

对 $L(s) = \dfrac{K_p\cdot K_0/\tau_{PI}}{s(T_{small}s+1)}$ 形式，MO 条件 $K_p\cdot K_0/\tau_{PI} = \dfrac{1}{2T_{small}}$，解得：

$$\boxed{K_{p,\omega}^{single} = \frac{\tau_{PI}}{2\cdot T_{small}\cdot K_0} = \frac{42.3\text{ ms}}{2\times 2.575\text{ ms}\times 798.0} = 0.01030\ \text{[占空比/(rad/s)]}}$$

$$\boxed{K_{i,\omega}^{single} = \frac{K_{p,\omega}^{single}}{\tau_{PI}} = \frac{0.01030}{42.3\text{ ms}} = 0.2436\ \text{[占空比/rad]}}$$

**单闭环性能预测（MO 法固有特性）**：

| 指标 | 数值 | 是否满足题目要求 |
|------|------|----------------|
| 闭环带宽 $\omega_b \approx 1/(\sqrt{2}T_{small})$ | 275 rad/s ≈ **43.7 Hz** | — |
| 阶跃超调 | **≈ 4.3%**（MO 法固有） | ✓ R2（<10%） |
| 等效闭环时间常数 $\tau_{cl} = 2T_{small}$ | 5.15 ms | — |
| 上升时间 $t_r \approx 2.2\tau_{cl}$ | **≈ 11.3 ms** | ✓ R1（≤0.8 s） |

> **注意**：单闭环的速度上升时间（理论上 11.3 ms）看起来比双闭环更快，是因为单闭环没有内环的"二阶动态叠加"。但代价是**电流完全不受控**，启动瞬间最大冲击电流可达 $U_{dc}/R_{eq} = 48/0.30 = 160$ A（远超 12 A 限额），**必须加电流截止保护**。

#### 5.4.4 单闭环专有设计：电流截止保护（不可省略）

双闭环依靠内环限幅天然实现电流保护；单闭环没有内环，**必须在 Simulink 中显式添加**：

**方案**：电流截止限速（Simulink 实现）

```matlab
% 在速度PI输出前，加电流监控截止逻辑
% Switch 块：|i_fb| > I_limit → 强制 D* = 0（切断驱动）
% 恢复条件：|i_fb| < I_limit 时，恢复正常 PI 输出
I_limit = 12;  % A，= 电机峰值电流

% 等效于在速度 PI 输出 D* 后串联一个判断：
% D_out = (abs(i_fb) < I_limit) ? D_PI : 0;
```

**Simulink 搭建方法**：
- `Abs` 块取电流绝对值 → `Compare To Constant`（>12）→ `Switch` 块控制 PI 输出通断
- 或：用 `Saturation Dynamic` 块，随电流动态限制 D* 的幅值

#### 5.4.5 完整参数录入（直接加入 `init_params.m`）

```matlab

% 被控对象：G_total = 798.0/[(42.3ms·s+1)(2.50ms·s+1)(75μs·s+1)]
% 注：单闭环Kp/Ki量级与双闭环完全不同，不能互换
```

#### 5.4.6 双闭环 vs 单闭环：完整量化对比表（用于报告 3.6 节）

| 对比项 | 双闭环（v1.4，MO+SO） | 单闭环（v1.4，MO+极点对消） | 优劣判断 |
|--------|------------------------|------------------------------|----------|
| **速度环 Kp_ω** | 14.14 [A·s/rad] | 0.01030 [占空比/(rad/s)] | — |
| **速度环 Ki_ω** | 5892 [A/rad] | 0.2436 [占空比/rad] | — |
| **电流环 Kp_i / Ki_i** | 0.1042 / 41.67 | 无 | — |
| **速度指令前置滤波** | τ_F = 2.40 ms（必须） | 无（不需要） | 单闭环更简单 |
| **被控对象类型** | 两级分离（MO→SO） | 完整三极点 + MO | — |
| **速度环带宽** | 265 Hz | 43.7 Hz | **双闭环快 6.1 倍** |
| **速度超调（理论）** | ~8%（含滤波） | ~4.3%（MO固有） | 单闭环略小 |
| **速度上升时间** | ~10 ms 级 | ~11 ms 级 | 双闭环略快 |
| **最大动态电流** | ≤ 12 A（内环限幅） | 理论冲击 160 A → 需截止 | **双闭环安全** |
| **抗负载扰动（20% 阶跃）** | 内环快速补偿，恢复快 | 速度环单独补偿，较慢 | **双闭环强** |
| **坡道下行制动** | 电流反向可控，平稳 | 反电势直接耦合，欠控 | **双闭环优** |
| **正反转切换** | 内环限流，过零平稳 | 冲击电流大，风险高 | **双闭环优** |
| **调参复杂度** | 4 参数 + 1 滤波 | 2 参数 | **单闭环简单** |
| **工程可靠性** | 高（工业标配方案） | 低（无原生限流） | **双闭环优** |

> **报告 3.6 节的对比结论写法建议（S5 参考）**：
> 
> 单闭环在参数简单性和理论超调上略有优势，但由于被控对象含反电势耦合且无电流内环，动态电流不可控，在坡道启动、正反转切换等工况中存在过流风险。双闭环通过引入电流内环，将电流动态与速度动态解耦，显著提升了抗扰能力和四象限运行可靠性，是工业级 BLDC 驱动系统的标配方案。仿真验证将对两种方案的动态响应指标（$t_r$、$\sigma$、抗扰恢复时间）和电流波形进行量化对比。
