## 第 0 章 · 本方案的阅读指南（必读）

本文档按"**分层设计 (Layered Design)**"组织，整个伺服驱动控制系统被分解为 **L1~L6 共 6 层**。每一层对应一位分系统工程师的责任边界、输入契约（上层传下来的量）、输出契约（本层必须交付给下层的量）。**分系统工程师只要遵守契约、不跨层改动，系统就能完整拼装**。

| 层号 | 层名                | 责任人               | 输入契约                             | 输出契约                                                             |
| ---- | ------------------- | -------------------- | ------------------------------------ | -------------------------------------------------------------------- |
| L1   | 工况与负载层        | S2                   | 题设 5 个机械参数 + 工况组合         | 阻力力矩$T_L(v)$、折算惯量 $J_{eq}$、工况时序表                  |
| L2   | 电机与传动层        | S2                   | L1 输出                              | 电机型号、减速比$i$、电机铭牌参数 $(R_s, L_s, K_e, K_t, J_m, p)$ |
| L3   | 主电路与电源层      | S3                   | L2 电机电气参数 + 48 V 母线          | 三相逆变桥拓扑、MOSFET 型号、母线电容、采样电路、保护门限            |
| L4   | 控制器层            | S4                   | L2 电机参数 + L3 采样带宽 + 性能指标 | 单闭环 / 双闭环 PI 参数、限幅值、开关频率                            |
| L5   | Simulink 集成建模层 | S4 主导 + S2/S3 配合 | L1~L4 全部输出                       | 可运行的 `.slx` 模型文件、`params.m` 参数脚本                    |
| L6   | 工况验证与报告层    | S5                   | L5 模型                              | 六大工况仿真数据 + 报告 + PPT                                        |

**关键约束（全员遵守）**：

1. **软件基线**：MATLAB / Simulink **R2023a**，仅使用三个工具箱：**Simulink、Simscape、Simscape Electrical**。**禁用** Motor Control Blockset、SimPowerSystems 遗留库、Stateflow 高级特性（避免 licence / 版本问题）。
2. **参数单一入口**：所有数值参数集中在 `init_params.m` 一个脚本里，模型中**严禁硬编码数字**（仅允许变量名引用）。这是 S5 批量跑工况的前提。
3. **标定方向**：以机器人**前进方向为正**；转速、电流、转矩、母线能量流向正负号在全模型前后一致。
4. **参考模板**：本项目以 MathWorks 官方示例 **"BLDC Motor Speed Control with Cascade PI Controllers"**（文档 ID: `slcontrol/ug/bldc-motor-speed-control-with-cascade-pi-controllers.html`）为**核心骨架模板**，在其上做"参数替换 + 工况扩展 + 单闭环对比分支"。**不从零搭建**。

### 2.2 为什么选 BLDC + 双闭环 PI + 三相 MOSFET 桥？（顶层决策仲裁）

| 决策点        | 候选                                       | 最终选择                                            | 仲裁理由                                                                                                                                                                                                                                                                                                                                               |
| ------------- | ------------------------------------------ | --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 电机类型      | 直流有刷 /**BLDC** / PMSM / 感应电机 | **BLDC（梯形反电势、霍尔换相）**              | ①48 V / 150 W 区间全球低压机器人轮毂电机**超 90% 是 BLDC**；②MathWorks 官方 Simscape Electrical 自带 `BLDC` 元件 + 参考模型 (`ee_bldc_speed_control.slx`, `BLDC Motor Speed Control with Cascade PI Controllers`)；③PMSM 需 FOC + SVPWM + Park 变换，**R2023a 标配工具箱下工作量 3 倍**；④有刷直流维护性差，不符合仓储连续作业。 |
| 传动机构      | 减速机 /**行星减速机** / 同步带      | **20:1 行星减速机（集成在轮毂内）**           | 轮毂电机标准配置；效率 0.90 与题设吻合；Simscape `Simple Gear` 可一键建模。                                                                                                                                                                                                                                                                          |
| 主电路拓扑    | H 桥 /**三相全桥逆变** / 半桥        | **三相 6-MOSFET 全桥逆变**                    | BLDC 强制三相；MOSFET 在 48 V / <20 A 场景**远优于 IGBT**（更低 $R_{DS(on)}$、更低开关损耗）。                                                                                                                                                                                                                                                 |
| 调速方式      | 线性调压 /**PWM 调压 + 六步换相**    | **PWM + 六步换相（120° 导通）**              | BLDC 最成熟方案；`Six-Step Commutation` 逻辑 20 行代码可实现；避免 SVPWM/FOC 的复杂度。                                                                                                                                                                                                                                                              |
| 控制结构      | 开环 / 单闭环 /**双闭环**            | **主用双闭环（外速度+内电流）+ 单闭环作对比** | 题目明确要求对比；双闭环能同时满足 R2 (速度超调) 和 R3 (电流超调) 指标；内环限流即是 R8 过载保护。                                                                                                                                                                                                                                                     |
| 位置/速度传感 | 霍尔 / 编码器 / 无感                       | **3 × Hall 霍尔传感器**                      | BLDC 标配；60°/120° 电角度分辨率已满足六步换相；成本极低；Simulink 中用 `Hall Sensor` block 或自定义角度门限实现。                                                                                                                                                                                                                                 |
| 电流传感      | 分流电阻 / 霍尔电流                        | **2 × 低边分流电阻 + 运放**                  | 48 V / <20 A 场景成熟方案（所有 BLDC 方案 IC 如 DRV8301、MP6540 均内置）。仿真中用理想电流计代替，不影响控制验证。                                                                                                                                                                                                                                     |
| 制动方案      | 回馈 /**能耗**                       | **能耗制动（斩波 + 制动电阻）为主，回馈为辅** | 48 V 电池系统回馈存在过充风险；能耗制动简单可靠；R2023a Simscape `Variable Resistor` + 母线电压检测即可实现。                                                                                                                                                                                                                                        |

#### 3.1.2 数学模型推导（核心，必须完整写进报告 3.1 节）

**【推导 1：单轮线速度与轮侧角速度】**

$$
v = n \cdot \frac{2\pi r}{60} \quad \Rightarrow \quad \omega_w = \frac{v}{r} = \frac{2\pi n}{60}
$$

代入：

- $n_{low}=60$ rpm → $\omega_{w,low} = 2\pi \approx 6.283$ rad/s → $v_{low} = 0.503$ m/s
- $n_{high}=240$ rpm → $\omega_{w,high} = 8\pi \approx 25.133$ rad/s → $v_{high} = 2.011$ m/s

**【推导 2：平路单轮阻力力矩】**

单轮承受的法向压力 $N_1 = \frac{m g}{2} = \frac{50 \times 9.81}{2} = 245.25$ N
单轮滚动阻力 $F_{r,1} = f \cdot N_1 = 0.04 \times 245.25 = 9.81$ N
轮侧阻力力矩：

$$
T_{L,\text{平},w} = F_{r,1} \cdot r = 9.81 \times 0.08 = 0.785 \text{ N·m}
$$

**【推导 3：5° 上坡单轮阻力力矩】**（最恶劣工况）

坡上法向压力与切向重力分量：

- $N_{1,\alpha} = \frac{mg}{2}\cos\alpha = 245.25 \times \cos 5° = 244.32$ N
- $F_{g,1} = \frac{mg}{2}\sin\alpha = 245.25 \times \sin 5° = 21.38$ N

单轮滚动阻力（考虑坡度）:

$$
F_{r,1,\alpha} = f \cdot N_{1,\alpha} = 0.04 \times 244.32 = 9.77 \text{ N}
$$

单轮总切向力（阻力 + 重力分量）：

$$
F_{\Sigma,1} = F_{r,1,\alpha} + F_{g,1} = 9.77 + 21.38 = 31.15 \text{ N}
$$

轮侧阻力力矩：

$$
T_{L,\text{坡},w} = F_{\Sigma,1} \cdot r = 31.15 \times 0.08 = 2.492 \text{ N·m}
$$

**【推导 4：减速机折算到电机轴侧 —— 选定减速比 i=20】**

选型依据（详见 3.2 节）：目标电机额定转速 ≈ 4800 rpm，故 $i = 4800/240 = 20$。

**电机轴侧稳态负载力矩**（考虑传动效率 $\eta_m = 0.90$）：

$$
T_{L,m} = \frac{T_{L,w}}{i \cdot \eta_m}
$$

- 平路稳态：$T_{L,m,\text{平}} = 0.785 / (20 \times 0.9) = 0.0436 \text{ N·m} = 43.6 \text{ mN·m}$
- 坡道稳态：$T_{L,m,\text{坡}} = 2.492 / (20 \times 0.9) = 0.1385 \text{ N·m} = 138.5 \text{ mN·m}$

**电机轴侧额定转速**：

$$
n_{m,\max} = n_{high} \cdot i = 240 \times 20 = 4800 \text{ rpm} \Rightarrow \omega_m = 502.65 \text{ rad/s}
$$

**【推导 5：转动惯量折算（机械轴 → 电机轴）】**

整机平动动能等于两个驱动轮的轮侧转动动能之和（能量守恒）：

$$
\frac{1}{2} m v^2 = 2 \cdot \frac{1}{2} J_{v,w} \omega_w^2, \quad v = \omega_w r
$$

解得**单轮承担的平动等效惯量**：

$$
J_{v,w} = \frac{1}{4} m r^2 = \frac{1}{4} \times 50 \times 0.08^2 = 0.08 \text{ kg·m}^2
$$

车轮自身转动惯量（估算为均匀圆盘 $m_w \approx 1.5$ kg）：

$$
J_w = \frac{1}{2}m_w r^2 = \frac{1}{2}\cdot 1.5 \cdot 0.08^2 = 4.8 \times 10^{-3} \text{ kg·m}^2
$$

折算到电机轴侧（除以 $i^2$）：

$$
J_{L,m} = \frac{J_{v,w} + J_w}{i^2} = \frac{0.08 + 0.0048}{20^2} = 2.12\times 10^{-4} \text{ kg·m}^2
$$

**电机自身惯量**（42BLF02 典型值 $J_m \approx 48$ g·cm² = $4.8 \times 10^{-6}$ kg·m²）：

**系统总折算惯量**：

$$
J_{eq} = J_m + J_{L,m} = 4.8\times10^{-6} + 2.12\times10^{-4} \approx 2.17\times10^{-4} \text{ kg·m}^2
$$

> **注**：由于 $J_{L,m} \gg J_m$（44 倍），本系统是**典型大负载惯量驱动系统**，对电流环响应速度要求较高。这个结论直接指导 L4 的 PI 带宽设计。

**【推导 6：动态过程所需最大电机轴力矩（R1 指标约束）】**

上升时间 $t_r \le 0.8$ s，从 0 加速到 $\omega_m=502.65$ rad/s，所需角加速度：

$$
\dot\omega_m = \omega_m / t_r = 502.65 / 0.8 = 628.3 \text{ rad/s}^2
$$

动态加速力矩：

$$
T_{\text{加速}} = J_{eq} \cdot \dot\omega_m = 2.17\times10^{-4} \times 628.3 = 0.1362 \text{ N·m}
$$

坡道最恶劣工况下电机轴所需**峰值力矩**（加速 + 克服坡道阻力）：

$$
T_{m,\text{peak}} = T_{\text{加速}} + T_{L,m,\text{坡}} = 0.1362 + 0.1385 = 0.275 \text{ N·m}
$$

**【推导 7：电机额定力矩、功率要求】**

稳态最恶劣（坡道巡航）：

$$
P_{m,\text{稳态}} = T_{L,m,\text{坡}} \cdot \omega_m = 0.1385 \times 502.65 = 69.6 \text{ W}
$$

**结论**：选用 **100 W 额定 / 200 W 峰值** 级别的 BLDC 电机，**留 50% 裕量即可满足稳态与 2× 过载（R8）要求**，符合题设"单轮电机连续功率上限 150 W，10 s 内允许 2 倍过载"。

### 4.3 关键参数计算

#### 4.3.1 PWM 频率选择

选 **$f_{PWM} = 20$ kHz**（主流 BLDC 驱动频率，超出可闻噪声，MOSFET 开关损耗可控）。

#### 4.3.2 母线电容

纹波电流有效值估算：BLDC 120° 导通，母线电流准方波，$I_{bus,rms} \approx 0.8 I_{phase,peak} = 0.8\times 10 = 8$ A 峰值 → 纹波按 5 A rms 估计。

$$
C_{bus} \ge \frac{I_{bus,rms}}{2\pi f_{PWM} \cdot \Delta U} = \frac{5}{2\pi \times 20000 \times 0.48} = 83 \text{ μF}
$$

取 **470 μF** 为 5× 裕量，且叠加陶瓷旁路抑制高频尖峰。

#### 4.3.3 制动电阻能量计算（工况⑤关键）

满载 5° 下坡从 240 rpm 减速到 0：

- 平动动能：$\frac{1}{2}m v_{high}^2 = 0.5 \times 50 \times 2.011^2 = 101.1$ J
- 转动动能：$2\times\frac{1}{2}J_{w}\omega_w^2 = 0.0048 \times 25.13^2 = 3.03$ J
- 下坡势能持续释放：$m g v_{high} \sin\alpha = 50 \times 9.81 \times 2.011\times 0.0872 = 86$ W 持续输入
- 摩擦损耗：$2 \times F_{r,1} \times v = 2\times 9.81\times 2.011 = 39.5$ W

制动过程按 1 s 估算，总吸能约 150 J，功率峰值约 200 W。

**制动电阻取 10 Ω / 50W 铝壳电阻**：

- 瞬时功率 $P_{R,peak} = U_{dc}^2/R = 48^2/10 = 230$ W，符合 200 W 峰值要求；
- 铝壳电阻短时过载能力强，50 W 额定 × 3~5 倍短时允许；
- 成本 ~¥ 15。

**斩波门限**：母线电压 $V_{dc} > 54$ V 时开启制动斩波（迟滞控制，回 50 V 关闭）。

## 第 5 章 · L4 层：控制器设计与 PI 整定（S4 主力）

> **本章对应题目任务 ②（控制方案 3 分）+ ⑥（建模/控制器/仿真 9 分）**

### 5.1 控制结构对比（题目明确要求对比）

#### 5.1.1 方案 A：单闭环（仅速度环）

```
ω* ──→ [─] ──→ [PI_ω] ──→ Duty ──→ [逆变+换相+电机+机械] ──→ ω
         ↑                                                      │
         └──────────────────────────────────────────────────────┘
```

- **优点**：结构简单、参数少（仅 $K_{p\omega}, K_{i\omega}$ 两个）、调试容易。
- **缺点**：启动/重载时电流无限幅 → 无法满足 R8 过载保护，无法实现 R3 电流超调 <5%；电流环是"看不见"的固有极点，动态响应受限。
- **在本项目中的地位**：**对照组**（报告中与双闭环对比分析，帮助论证双闭环必要性）。

#### 5.1.2 方案 B：双闭环（外速度环 + 内电流环）—— 本方案**主用**

```
       ω*          i_q*                     Duty
ω* ──→[─]──→[PI_ω]──→[限幅±I_max]──→[─]──→[PI_i]──→[换相+逆变+电机+机械]──→ω
       ↑                              ↑                │                     │
       └─────────────── ω 反馈 ───────┼────────────────┘                     │
                                       └──────── i_q 反馈 ────────────────── │
                                                                              │
    (i_q 反馈：由两相分流电阻采样，结合换相相位重构为等效 q 轴电流)
```

- **优点**：内环天然限流 = R8 过载保护 + R3 电流超调可控；外环只管转速，动态快；抗扰能力强（扰动主要进内环）。
- **缺点**：两套 PI 4 个参数；需带宽分离（内环带宽 > 外环 5~10 倍）。
- **对 BLDC 的简化**：由于六步换相下每一刻只有两相导通，**"等效 q 轴电流"≈ 母线电流 ≈ 两相电流中非零者**。可用 `max(abs(i_a), abs(i_b), abs(i_c))` 简化，避免引入 Park 变换。

### 5.2 数学模型（控制器设计的基础，写进报告 3.6 节）

#### 5.2.1 BLDC 两相导通等效直流电机模型

BLDC 在 120° 导通模式下，每时刻两相串联通电，等效为一台两相绕组串联的"直流电机"：

**电压方程**：

$$
U_{dc} \cdot D = 2 R_s \cdot i + 2 L_s \frac{di}{dt} + E
$$

其中 $E = K_e' \cdot \omega_m$，线-线反电势常数 $K_e' = K_e \sqrt{3}$（峰值换算）；$D$ 为 PWM 占空比。

**转矩方程**：

$$
T_e = K_t' \cdot i
$$

**机械方程**：

$$
T_e - T_L = J_{eq} \frac{d\omega_m}{dt} + B\omega_m
$$

其中 $B$ 为摩擦阻尼系数（估算 $B \approx T_N / (10 \omega_N)$，或直接使用铭牌空载电流对应的损耗力矩折算）。

#### 5.2.2 关键传递函数

**电流环被控对象**（忽略反电势近似，因内环带宽 >> 机械时间常数）：

$$
G_i(s) = \frac{I(s)}{U(s)} = \frac{1/(2R_s)}{\tau_e s + 1}, \quad \tau_e = \frac{2L_s}{2R_s} = \frac{L_s}{R_s} = \frac{0.75\text{ mH}}{0.6\,\Omega} = 1.25 \text{ ms}
$$

**速度环被控对象**（电流环视为理想带宽足够，内环等效为 1 阶 $\approx 1$）：

$$
G_\omega(s) = \frac{\Omega(s)}{I(s)} = \frac{K_t}{J_{eq} s + B} \approx \frac{K_t}{J_{eq} s}
$$

### 5.3 PI 参数整定（给出分析解 + 实测微调两步法）

#### 5.3.1 内电流环 PI（用"零极点对消"方法 —— 工程最可靠）

PI 控制器 $C_i(s) = K_{p,i}\left(1 + \frac{1}{\tau_i s}\right) = K_{p,i}\frac{\tau_i s + 1}{\tau_i s}$

**对消策略**：令 $\tau_i = \tau_e = 1.25$ ms，即 $K_{i,i} = K_{p,i}/\tau_i$。

对消后开环传递：$C_i G_i = \frac{K_{p,i}/(2R_s)}{\tau_i s}$

闭环带宽 $\omega_{bi}$：目标选 **$\omega_{bi} = 2\pi \times 1000 = 6283$ rad/s**（约为 PWM 频率 20 kHz 的 1/20，工程经验值）。

闭环后一阶环节 $\frac{1}{\tau_i \cdot 2R_s / K_{p,i} \cdot s + 1}$，令其时间常数 $= 1/\omega_{bi}$：

$$
K_{p,i} = \omega_{bi} \cdot \tau_i \cdot 2R_s = 6283 \times 1.25\times10^{-3} \times 1.2 = 9.42
$$

$$
K_{i,i} = K_{p,i}/\tau_i = 9.42 / 1.25\times10^{-3} = 7536
$$

（注：量纲为 电压/电流，即 V/A。实际 Simulink 中 PI 输出是占空比 D，需除以母线电压 $U_{dc}$ 归一化，等价于 $K_{p,i}' = 9.42/48 = 0.196$，$K_{i,i}' = 157$）

#### 5.3.2 外速度环 PI（用"带宽分离"方法）

内环闭环视为一阶惯性 $G_{i,cl}(s) \approx \frac{1}{1/\omega_{bi} \cdot s + 1}$，对速度环等效为近似 1。

速度环开环：$C_\omega G_\omega \approx K_{p,\omega}\left(1+\frac{1}{\tau_\omega s}\right)\cdot \frac{K_t}{J_{eq} s}$

目标速度环带宽 $\omega_{b\omega} = \omega_{bi}/10 = 628$ rad/s（100 Hz，工程经验）。

经典整定（幅值裕度 > 6 dB，相角裕度 > 60°）：

$$
K_{p,\omega} = \omega_{b\omega} \cdot J_{eq} / K_t = 628 \times 2.17\times 10^{-4} / 0.085 = 1.60
$$

$$
\tau_\omega = 10/\omega_{b\omega} = 10/628 = 15.9 \text{ ms}
$$

$$
K_{i,\omega} = K_{p,\omega}/\tau_\omega = 1.60/0.0159 = 100.6
$$

#### 5.3.3 PI 参数整定分步法（S4 按此步骤执行，S5 归档过程）

| 步骤                                | 动作                                                   | 期望现象                                               | 若不满足怎么办                                                          |
| ----------------------------------- | ------------------------------------------------------ | ------------------------------------------------------ | ----------------------------------------------------------------------- |
| **Step 1** 内环孤立调试       | 断开外环，手动给$i^*=2$ A 阶跃                       | 电流上升时间 <2 ms，超调 <5%                           | $K_{p,i}$ 每次乘/除 1.5 倍迭代                                        |
| **Step 2** 内环带限幅         | 加入$\pm I_{peak}=10$A 限幅 + 抗积分饱和（clamping） | 饱和时积分不爆炸                                       | 开 Anti-windup 的 `back-calculation` 模式                             |
| **Step 3** 外环加入，空载阶跃 | 从 0 阶跃到 240 rpm 空载                               | $t_r < 0.3$ s, $\sigma_n < 10\%$                   | $K_{p,\omega}$ 微调 ±20%，$\tau_\omega$ 微调 ±50%                 |
| **Step 4** 额定坡道阶跃       | 满载坡道 0→240 rpm                                    | **$t_r \le 0.8$ s**（R1），$\sigma_n < 10\%$ | 若$t_r$ 超标：加大 $K_{p,\omega}$；若超调超标：加大 $\tau_\omega$ |
| **Step 5** 抗扰测试           | 稳态 240 rpm 加 20% 负载阶跃                           | 稳态误差 → 0（5τ 内恢复）                            | 若偏差残留：加大$K_{i,\omega}$                                        |
| **Step 6** 四象限             | 正反转、制动切换                                       | 无失控、电流不超限                                     | 检查换相逻辑、制动斩波门限                                              |

### 6.3 关键子系统实现注意事项

**Mechanical Load 子系统（S2 交付）**：

- 输入：电机轴转速 $\omega_m$、坡度信号 $\alpha(t)$、载荷 $m(t)$
- 输出：电机轴侧负载力矩 $T_{L,m}$
- 内部实现：按 3.1.2 节公式写 Simulink 函数块 `Fcn` 或 MATLAB Function
- **验收**：单独跑仿真，给定 $\omega_m$ 斜坡输入，输出 $T_{L,m}$ 应与手算吻合

**Power Stage 子系统（S3 交付）**：

- 首轮：平均模型 = 受控电压源 × 占空比，节省仿真时间；
- 第二轮：替换为 Simscape MOSFET 开关模型，验证纹波与保护动作。

**Controller 子系统（S4 交付）**：

- 使用 Simulink `Discrete PID Controller` block，参数 Back-calculation anti-windup；
- 采样时间 $T_s = 50\mu s$；
- 额外输出 `i_ref` 供调试观察。

### 7.1 工况脚本模板（S5 按此扩展成 6 个）

```matlab
%% run_case_02_loaded.m  —— 额定负载坡道启动
run('init_params.m');
slope_deg = 5;            % 坡度

omega_ref_rpm = 240;      % 目标轮速
sim_time = 2.0;

% 运行仿真
simOut = sim('bldc_diffdrive_top', sim_time);

% 提取关键波形
t    = simOut.logsout.get('omega_wheel').Values.Time;
w    = simOut.logsout.get('omega_wheel').Values.Data * 60/(2*pi);  % rpm
iq   = simOut.logsout.get('i_fb').Values.Data;
Te   = simOut.logsout.get('T_e').Values.Data;
Vdc  = simOut.logsout.get('V_dc').Values.Data;

% 性能指标计算
rise_time = compute_rise_time(t, w, 0.9*omega_ref_rpm);
overshoot = (max(w) - omega_ref_rpm) / omega_ref_rpm * 100;
steady_err = mean(w(end-1000:end)) - omega_ref_rpm;

fprintf('Case 2: tr=%.3fs, σ=%.1f%%, ess=%.2f rpm\n', rise_time, overshoot, steady_err);

% 出图
plot_results(t, w, iq, Te, Vdc, 'Case 2 - Loaded Slope');
```
